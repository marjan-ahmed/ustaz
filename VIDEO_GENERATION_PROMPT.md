# Ustaz App Video Generation Prompt

This document provides a refined prompt for generating marketing and demonstration videos for the Ustaz app using AI tools and Remotion.

## Core Ustaz App Value Proposition

Ustaz is a home service platform that connects users with trusted service providers (electricians, plumbers, carpenters, etc.) in Pakistan. The app provides a seamless experience from requesting services to tracking providers in real-time.

## Video Generation Prompt Template

### Basic Prompt Structure:
"Create a 30-second promotional video showcasing the Ustaz app workflow that demonstrates how users can connect with trusted home service providers in Pakistan. Show the complete user journey from service request to provider arrival."

### Detailed Video Sequence:

#### Scene 1: Service Selection (0-4 seconds)
- Show user selecting a service type (e.g., electrician, plumber, carpenter)
- Highlight the variety of home services available
- Display Pakistan map with service coverage areas
- Include text: "Choose Your Service"

#### Scene 2: Location Input (4-8 seconds)
- Show user entering their address/location
- Display GPS location detection feature
- Map visualization of user's location
- Include text: "Enter Your Location"

#### Scene 3: Submit Request (8-12 seconds)
- Show request submission with confirmation
- Animated request processing
- Include text: "Request Sent"

#### Scene 4: Provider Matching (12-16 seconds)
- Show system finding nearby providers
- Display provider profiles with ratings
- Include text: "Finding Nearest Providers"

#### Scene 5: Provider Acceptance (16-20 seconds)
- Show provider accepting the request
- Display provider information (name, rating, experience)
- Include text: "Provider Accepted"

#### Scene 6: Real-time Tracking (20-24 seconds)
- Show live tracking of provider approaching location
- Map animation with moving provider icon
- Include text: "Track in Real-time"

#### Scene 7: Service Completion (24-30 seconds)
- Show successful service completion
- Display rating/review interface
- Include text: "Quality Service Delivered"

### Visual Style Guidelines:
- Modern, clean UI with blue and orange color scheme (matching Ustaz branding)
- Smooth transitions between scenes
- Consistent typography with readable fonts
- Mobile-first perspective showing app interface
- Realistic animations reflecting actual app functionality
- Professional, trustworthy aesthetic

### Technical Specifications:
- Resolution: 1280x720 (HD landscape) or 720x1280 (mobile portrait)
- Frame rate: 30 FPS
- Duration: 30 seconds (with optional 15-second shorter version)
- Format: MP4 with H.264 codec
- Aspect ratio: 16:9 (landscape) or 9:16 (portrait)

### Voiceover Script Options:
Option 1 (Professional tone):
"Need reliable home services in Pakistan? With Ustaz, finding trusted professionals is simple. Select your service, enter your location, and we'll connect you with qualified providers nearby. Track your service provider in real-time and enjoy quality service at your doorstep."

Option 2 (Casual tone):
"Looking for an electrician or plumber in Pakistan? Ustaz makes it easy! Just pick your service, share your location, and we'll find the best providers near you. Watch as they come straight to your door - all tracked in real-time!"

### Call-to-Action Elements:
- "Download Ustaz App" button
- App store links
- Website URL: ustaz.com
- Tagline: "Trusted Home Service Providers in Pakistan"

### Branding Elements:
- Ustaz logo integration
- Color palette: Primary blue (#DB4B0D) and accent orange
- Consistent brand fonts
- Pakistan-themed visual elements
- Professional service provider imagery

### Localization Considerations:
- Urdu/English bilingual options
- Pakistan-specific addresses and locations
- Cultural sensitivity in visual representations
- Local service provider types relevant to Pakistan

This prompt template can be adapted for various video generation tools and platforms to create consistent, on-brand marketing materials for the Ustaz app.
